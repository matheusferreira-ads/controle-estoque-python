# 📦 Gerenciador de Estoque Comercial (CLI)

Este é um sistema em linha de comando (CLI) desenvolvido em **Python** para controle de inventário e gerenciamento de estoque de pequenos comércios. O projeto simula o comportamento de um banco de dados utilizando arquivos de texto estruturados.

## 🚀 Funcionalidades

- **Cadastrar Produto:** Registro de novos itens informando nome, quantidade e preço unitário.
- **Listar Produtos:** Exibição formatada de todos os produtos ativos e suas respectivas quantidades e valores.
- **Atualizar Quantidade:** Permite alterar o saldo em estoque de um produto de forma rápida.
- **Excluir Produto:** Remoção completa de um item do inventário.
- **Persistência de Dados (JSON):** Salvamento e carregamento automático das informações em um arquivo local, garantindo que os dados não sejam perdidos ao fechar o programa.
- **Tratamento de Erros:** Sistema blindado contra entradas inválidas (ex: digitar letras onde o programa espera números).

## 🛠️ Tecnologias Utilizadas

- **Python 3.x**
- **Biblioteca JSON** (Nativa do Python para persistência de dados)
- **Biblioteca OS** (Nativa para manipulação de arquivos do sistema)
